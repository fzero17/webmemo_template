# My Jekyll Theme

A simple and clean Jekyll theme.

## Installation

Add this line to your Jekyll site's Gemfile:

```ruby
gem "webmemo_template"
```

And add this line to your Jekyll site's `_config.yml`:

```
theme: my-jekyll-theme
```

And then execute:

```
bundle install
```
